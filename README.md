# inGaia Font Icons

## Instalação

### npm

```console
$ npm install ingaia-font-icons --save
```

### Bower

```console
$ bower install ingaia.font.icons --save
```

## Adicionar mais ícones

1. Copie o arquivo SVG nas pasta SVG.
2. No console execute os comandos:

```console
$ npm install
```

```console
$ gulp
```
